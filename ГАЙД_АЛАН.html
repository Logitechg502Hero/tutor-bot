Открой бота в Telegram и напиши один раз:

Бот покажет панель управления. Потом просто /start.

Шаг 3 — Модерация заявок

Нажми 📋 Заявки на модерацию. Бот показывает анкету: фото, имя, предмет, опыт, описание, контакты, цена. Вверху счётчик: "Заявка 2 из 5".

  • Одобрить → анкета выйдет в канал в 12:00 или 19:00. Пользователь получит уведомление ✅
  • Отказать → бот попросит причину → пользователь получит сообщение с причиной ❌
  • Листать анкеты: кнопки ◀️ ▶️

Автоматическое одобрение: если в анкете нет ссылок — бот одобряет сам, без тебя.

Если пользователь добавил ссылку (300 ₽): он платит на +79857770845 (Тинькофф) и присылает скриншот. Скриншот придёт тебе в личку. Проверь перевод → зайди в Заявки → одобри вручную.

Шаг 4 — Найти пользователя

Нажми 🔍 Найти пользователя → введи Telegram ID (числовой, например 123456789).

  • Добавить пост — запланировать публикацию в конкретное время
  • Удалить пользователя — удалить спам-аккаунт

Чтобы узнать ID пользователя — пусть напишет боту /id.

Шаг 5 — Платные посты

Пользователь нажимает "💰 Купить посты" → бот говорит написать @Ts_alan. Ты договариваешься, получаешь деньги, затем:

  1. Нажми 🔍 Найти пользователя → введи его ID
  2. Нажми Добавить пост
  3. Введи дату и время: 15.06.2026 18:00
  4. Бот опубликует анкету автоматически в это время

Шаг 6 — Рассылка

Нажми 📢 Рассылка → выбери аудиторию:

  • 👥 Всем — и репетиторам, и ученикам
  • 🧑‍🏫 Только репетиторам
  • 🎓 Только ученикам

Напиши текст → бот покажет превью и количество → нажми ✅ Отправить.
После рассылки покажет: "Отправлено: 20 / Ошибок: 1".

Шаг 7 — Дашборд (веб-панель)

Открывается в браузере. Ссылку берёшь в Railway: Settings → Networking.

Войти: логин admin, пароль changeme

Видишь: карточки со статистикой + таблицу последних пользователей. Обновляется каждые 60 секунд.

Частые проблемы

Бот не отвечает
Railway → Deployments → если красная ошибка → нажми Redeploy.

Анкета не выходит в канал
Зайди в Заявки — возможно ждёт одобрения. Одобри вручную.

Нужно опубликовать срочно прямо сейчас
Найди пользователя → Добавить пост → поставь время на ближайшую минуту.

Пользователь застрял в анкете
Пусть нажмёт ❌ Отмена или напишет /start.
